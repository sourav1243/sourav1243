I build ML systems focused on NLP and knowledge graphs. Most of my current work is around Retrieval-Augmented Generation with medical domain applications. Right now I'm exploring AI self improvement capability.

## What I Work On

- Train and evaluate transformer-based models for NLP and RAG tasks end to end
- Build data pipelines and run experiments tracked with MLflow and versioned with DVC
- Serve models as REST APIs with FastAPI and containerize with Docker for reproducibility
- Construct knowledge graphs in Neo4j for hybrid retrieval and fact-checking systems
